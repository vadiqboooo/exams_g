# Обновление вкладки "Сотрудники"

## Изменения

### Переименование
- ✅ Вкладка "Учителя" → "Сотрудники"
- ✅ Все связанные тексты обновлены

### Функционал

**Теперь отображаются ВСЕ сотрудники:**
- Owner (владелец системы)
- School Admin (администраторы школ)
- Teacher (учителя)

### Роли при создании

В форме создания сотрудника доступны 3 роли:
1. **Учитель** - обычный учитель
2. **Администратор школы** - привязан к школе
3. **Владелец** - полные права (создавайте осторожно!)

### Защита

- ❌ Нельзя удалить владельца (owner)
- ❌ Нельзя удалить сотрудника с группами

## Что изменилось

### Frontend
- `App.jsx` - название вкладки "Сотрудники"
- `TeacherList.jsx` - заголовки, сообщения
- `TeacherForm.jsx` - добавлена роль "Владелец"

### Backend
- `GET /teachers/` - возвращает всех сотрудников
- `DELETE /teachers/{id}` - защита от удаления owner
- Все сообщения обновлены

## Использование

1. Перейдите на вкладку **"Сотрудники"** (доступна только owner)
2. Увидите список всех пользователей системы
3. Можете добавить новых сотрудников любой роли
4. Можете редактировать/удалять (кроме owner)

## Таблица отображает:
- ID
- Имя пользователя
- Имя сотрудника
- Роль (Владелец/Админ школы/Учитель)
- Школа (для админов школ)
- Действия (Редактировать/Удалить)

## Важно

⚠️ **Не создавайте несколько владельцев без необходимости!**
Роль "Владелец" дает полные права на систему.
